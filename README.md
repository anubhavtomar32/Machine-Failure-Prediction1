# Machine Failure Prediction 🚀

This project predicts machine failure using Machine Learning.

## 🔧 Features
- Predicts failure based on sensor data
- Random Forest ML model
- Flask Web App

## 📊 Dataset
Includes parameters like:
- Temperature
- Vibration
- Pressure
- Runtime
- Humidity

## ▶️ How to Run

```bash
pip install -r requirements.txt
python model.py
python app.py
```
